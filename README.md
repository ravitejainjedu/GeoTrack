# GeoTrack

Real-time telemetry tracking system with .NET backend and React frontend.

## Components
- **API**: ASP.NET Core Web API with SignalR and PostgreSQL
- **UI**: React with Leaflet maps
- **Simulator**: Node.js telemetry generator

## Setup
1. `docker compose up --build`
2. Access UI at http://127.0.0.1:5173
